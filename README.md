# Hermes Agent - Home Assistant App Repository

This repository provides the unofficial Home Assistant App for **Hermes Agent** by Nous Research. 
Hermes is a self-improving AI agent with a built-in learning loop and an API/messaging gateway.

## 🛠️ How to Install

1. Open your Home Assistant instance.
2. Navigate to **Settings** > **Apps** > **App Store**.
3. Click the three dots (top right) and select **Repositories**.
4. Add the following URL: `https://github.com/dartanidi/ha-hermes-agent`
5. Click **Add**, then close the dialog.
6. Search for "Hermes Agent" in the list, click it, and press **Install**.
7. Once installed, go to the "Configuration" tab to set your API key, then **Start** the App.

## 🖥️ Integrated Web Terminal
This App includes an integrated web terminal (via Ingress) to perform the initial setup and interact with the agent's CLI directly from the Home Assistant sidebar.
