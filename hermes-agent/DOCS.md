# Setup Instructions

After starting the App for the first time, you need to configure your LLM providers (e.g., Ollama, OpenAI) and messaging platforms.

### Step 1: Initial Setup
1. Enable the **"Show in sidebar"** toggle in the App info page.
2. Click on the **Hermes Terminal** icon in your Home Assistant sidebar.
3. In the terminal, type the following command to start the interactive wizard:
   ```bash
   hermes setup
